🌧️ Rain Alert System using Python

A simple Python project that checks weather forecast data and sends an SMS alert if rain is expected — so you never forget your umbrella ☔

🚀 Features
Fetches real-time weather forecast using OpenWeather API
Detects rain conditions automatically
Sends SMS alerts using Twilio API
Uses environment variables to keep credentials secure 🔐
🛠️ Tech Stack
Python
requests (API calls)
Twilio API (SMS service)
OpenWeatherMap API
Environment Variables (os, .env)
